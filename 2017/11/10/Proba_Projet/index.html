    4 Key ideas in the article

    Futrell: Quantifying word order freedom in dependency corpora

    Bane: Quantifying morpho complexity

    (1) Kolmogorov complexity

    • One object is more complex than another if it takes longer to describe
    • In morphology, we say DL(x) = description length of a string x

    (2) Morphological Complexity Metric
    = (DL(affixes) + DL(signatures)) / (DL(affixes) + DL(sig) + DL(stems)

    (3) Corpora used in article

    • 14 languages by corpora traslation of the Bible
      https://www.biblegateway.com/versions/
    • 6 languages are creoles or pidgins gathered from Scannell’s 2007 automatic, language targeting web-crawler

    (4) Apperent trend

    • While the token count increases, the type count decrease, so as the complexity metric
    • Plausible Explanation: Distinctions or constructions that are expressed in languagues like Latin or Hungarian by morphologically altering the shapes of the words (token type high), are often expressed in morphologically simpler languagues by the presence of function words or particular syntactic arrangements, each instance of which represents new tokens of existing types (type token low).

    (5) Linguistica is in the core of this metric

    • Project Website:
      http://people.cs.uchicago.edu/~jagoldsm/linguistica-site/
    • GitHub Page:
      https://github.com/linguistica-uchicago/lxa5
    • Download as library and full doc: (many useful functions !!!)
      http://linguistica-uchicago.github.io/lxa5/lexicon.html
