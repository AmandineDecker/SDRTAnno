    小点积累

    (1)list += ‘hi’和list.append(‘hi’)区别：
    a = []
    a += ‘hi’ ==> a=[‘h’,’i’]
    a.append(‘hi’) ==> a=[‘hi’]
    (2)set.add()
    set是无序的，initiate时只能敲入一个argu并且arg必须是str，之后用add(‘’)来加入，可以sorted，一旦sorted转换为list
    a = set(‘a’) ==> {‘a’}
    a.add(‘b’) ==> {‘b’,’a’}
    a = sorted(a) ==> [‘a’, ‘b’]
